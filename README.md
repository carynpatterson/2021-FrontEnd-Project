This project is a simple score tracking website for the card game Sushi Go!. The website allows you to choose the number of players (2-5), input their names, their scores for each round, and the final scores for puddings. The calculate button will total each player's final score. 


CSS Features List: The flexbox changes from row alignment on desktop to column alignment for mobile.

JavaScript Features List: Javascript is used to calculate the player's scores and displays the total in page.

JavaScript Features List: The player dropdown box shows or hides additional player score cards based on the number of players selected using Javascript.
